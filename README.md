## Usage
### Set up
```shell
$ brew tap LemonNekoGH/tap https://github.com/LemonNekoGH/tap
```
### Install softwares
After setup, you can install softwares from the [projects list](#Projects).
```shell
$ brew update
$ brew install easi-gen
# or avoid clash
$ brew install LemonNekoGH/tap/easi-gen
```
